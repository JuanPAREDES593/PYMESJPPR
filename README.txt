Coké - Sitio web (MVP)
Archivos:
- index.html
- style.css
- script.js
- img/ (6 imágenes de ejemplo)

Instrucciones:
1. Descomprime el zip.
2. Reemplaza las imágenes en la carpeta img/ por tus fotos reales (mantén los nombres o actualiza el HTML).
3. Abre index.html localmente para probar.
4. Sube todos los archivos a tu repositorio de GitHub y activa GitHub Pages (Settings -> Pages -> main / root).
